A_yoga_I

Hey there! Welcome to A_yoga_I, developed in 2022 by Francis Monari.

I created A_yoga_I inspired by various Machine Learning models for visual recognition as Hackton Project. This is my first project with Machine Learning model and attempt at Web Development.
A_yoga_I is a virtual yoga trainer. It monitors your posture using Tensorflow during various yoga poses, and tells if you are doing it correctly or not. It has a button to learn so, first we can learn yoga poses within 10 seconds. After learning, we start doing practice poses within 30 seconds. There is another button about Philosophy of Yoga, it is a short description of it. where you can find words to click that are links to various websites, wikipedia and youtube videos. Feel free to watch and read about Philosophy of Yoga, it will give you better understanding and knowledge.

A_yoga_I uses Pose.Net, developed by Google’s TensorFlow, and ml5.js, a JavaScript library for Machine Learning on the browser. Most of my learning was following The Coding Train’s YouTube playlist on Pose Estimation.
